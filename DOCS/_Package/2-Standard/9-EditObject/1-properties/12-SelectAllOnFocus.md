# {{ page.title }}


Specifies the text selection state when the object receives focus.

 

In the case of $ STD , the specified operation ( select all ) is performed. In the case of $ NONE , the full selection operation is not performed.

The initial value is $ STD .

 

Valid for the following EditObject- derived classes:

<a href="/package/standard/textbox">TextBox</a>, <a href="/package/extension3/dateedit">DateEdit</a>, <a href="/package/extension3/numberedit">NumberEdit</a>, <a href="/package/extension4/maskedit">MaskEdit</a>
