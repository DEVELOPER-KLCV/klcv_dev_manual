# {{ page.title }}



Set whether to display the caret indicating the input position.

 

Specify $ TRUE if you want to show the caret, or $ FALSE if you don't want to show it .

The initial value is $ TRUE .

<br><small><span style="color:red">Added since Ver.5.0.3</span></small>
<br><small><span style="color:blue">Not supported in Mobile, AI</span></small>
