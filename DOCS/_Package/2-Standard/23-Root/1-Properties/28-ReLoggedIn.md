# {{ page.title }}
<br>

True if the Biz / Browser AI process is automatically restarted after being killed by an external factor.

The initial value is $ FALSE.

The value is reset when Biz / Browser is initialized by the <a href="/package/standard/root/methods/login">Login</a> method.

The initial value is $ FALSE.

<br><small><span style="color:red">Can only be used with AI</span></small>
