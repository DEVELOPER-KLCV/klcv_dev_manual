# {{ page.title }}

Performs horizontal alignment. $ STD is centered.

|Constant | Value | Description |
|---------|-------|-------------|
|$STD     | 0     |Same as $CENTER |
|$TOP     | 1     |Top alignment |
|$CENTER  | 2     |Center alignment |
|$BOTTOM  | 3     |Bottom alignment |
