# {{ page.title }}
<br>

Performs vertical alignment. $ STD is centered.

This property is valid only when printing. The screen is always centered.

| Constant  | Value | Description     |
|---------- |:-----:|-----------------|
| $STD      |   0   | Same as $ CENTER  |
| $TOP      |   1   | Top justified  |
| $CENTER   |   2   | Centered        |
| $BOTTOM   |   3   | Bottom justified |
