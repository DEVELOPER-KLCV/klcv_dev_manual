# {{ page.title }}
<br>

Selection mode. Select row by row, column by column, or cell by cell.

| Constant | Value | Description      |
|----------|:-----:|------------------|
| $STD     |   0   | Select by cell   |
| $COLUMN  |   1   | Select by column |
| $ROW     |   2   | Select by row    |
