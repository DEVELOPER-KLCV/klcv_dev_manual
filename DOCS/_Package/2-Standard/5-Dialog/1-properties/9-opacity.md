# {{ page.title }}

Specifies the transparency of the dialog.

Specify in the range of 0.0 (transparent) to 1.0 (opaque) .

 

The initial value is 1.0 (opaque) 

<small>Added since Ver.5.0.3 <br> Not supported in Mobile, AI</small>
