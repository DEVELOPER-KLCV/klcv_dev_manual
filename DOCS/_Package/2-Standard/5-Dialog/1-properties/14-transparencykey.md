# {{ page.title }}
 

Specifies the target color for the transparency effect.

 

In the case of $ STD , the transparency effect by specifying the color is not applied. If it is not $ STD , the transparency effect will be applied to the specified color.

 

The initial value is $ STD .

<small>Added since Ver.5.0.3 <br> Not supported in Mobile, AI</small>
