# {{ page.title }}

Performs vertical alignment. $STD is centered.

| Constant | Value | Description      |
|----------|:-----:|------------------|
| $ STD    |   0   | Same as $ CENTER |
| $ TOP    |   1   | Top alignment    |
| $ CENTER |   2   | Centered         |
| $ BOTTOM |   3   | Bottom alignment |
