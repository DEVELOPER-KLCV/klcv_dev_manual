# {{ page.title }}

The following methods are defined in the DragSource class.

|Name       | Description                 |
|-----------|-----------------------------|
|[AddFile](/package/extension4/dragsource/methods/addfile) |Set the file to drag and drop |
|[DoDragDrop](/package/extension4/dragsource/methods/dodragdrop) |Start drag and drop <br><small><span style="color:green">It is possible to specify both copy mode and move mode together since Ver.5.0.0</span></small>|
|[SetObject](/package/extension4/dragsource/methods/setobject) |Set the object to drag and drop |
|[SetString](/package/extension4/dragsource/methods/setstring) |Set the character string data to be dragged and dropped |
