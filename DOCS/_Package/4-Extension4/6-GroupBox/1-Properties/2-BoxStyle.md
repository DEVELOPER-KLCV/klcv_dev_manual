# {{ page.title }}

Specifies the type of group box.

| Constant  | Value | Description                   |
|-----------|:-----:|-------------------------------|
| $BOX      |   0   | Box type                      |
| $ROUND    |   1   | Box type with rounded corners |
| $HORZLINE |   2   | Horizontal line type          |
| $VERTLINE |   3   | Vertical line type            |
