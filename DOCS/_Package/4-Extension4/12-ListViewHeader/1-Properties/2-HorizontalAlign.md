# {{ page.title }}

Position alignment in column. $STD is left-aligned.

| Constant | Value | Description   |
|----------|:-----:|---------------|
| $STD    |   0   | Same as $LEFT |
| $LEFT   |   1   | Left-aligned    |
| $CENTER |   2   | Centered      |
| $RIGHT  |   3   | Right-aligned   |

<br><small><span style="color:green">It also affects the position of column names since Ver.5.0.0.</span></small>
