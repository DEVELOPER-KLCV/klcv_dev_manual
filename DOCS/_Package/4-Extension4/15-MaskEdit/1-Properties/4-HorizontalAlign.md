# {{ page.title }}

Align horizontally. $STD is left aligned.

| Constant | Value | Description   |
|----------|:-----:|---------------|
| $STD    |   0   | Same as $LEFT |
| $LEFT   |   1   | Left-aligned    |
| $CENTER |   2   | Centered      |
| $RIGHT  |   3   | Right-aligned   |
