# {{ page.title }}
<br>

Align horizontally. $STD is right aligned.

| Constant | Value | Description    |
|----------|:-----:|----------------|
| $STD     |   0   | Same as $RIGHT |
| $LEFT    |   1   | Left aligned   |
| $CENTER  |   2   | Centered       |
| $RIGHT   |   3   | Right aligned  |
